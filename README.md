# FocusLoop

FocusLoop is a gamified habit tracking web app and installable PWA designed to help users build consistency through streaks, XP progression, and weekly reflection analytics.

## Project Description

FocusLoop helps users develop consistent habits by combining tracking with gamification. Users can log daily activities, build streaks, earn XP, and analyse their progress through visual insights.

The purpose of this project is to demonstrate full-stack development skills, including backend API creation, database integration, authentication, and responsive frontend design.

## Features
- Habit tracking with custom tracks
- Streak system and XP progression
- Weekly analytics and insights engine
- Pomodoro focus timer
- Achievements and milestones
- Installable Progressive Web App (PWA)
- Dark mode and customizable themes

## Tech Stack
Frontend:
- HTML
- CSS
- Vanilla JavaScript
- Chart.js

Backend:
- Node.js
- Express.js

Database:
- MongoDB Atlas

Deployment:
- Render

## Usage
- Register or log in to your account
- Create habit tracks (e.g., Gym, Study, Reading)
- Log sessions daily to build streaks
- Earn XP and unlock achievements
- View analytics and insights in the dashboard
- Install the app for a native experience (PWA)
