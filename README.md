# FocusLoop

FocusLoop is a gamified habit tracking web app and installable PWA designed to help users build consistency through streaks, XP progression, and weekly reflection analytics.

## Live Demo
https://focusloop-alex.onrender.com

## Features
- Habit tracking with custom tracks
- Streak system and XP progression
- Weekly analytics and insights engine
- Pomodoro focus timer
- Achievements and milestones
- Installable Progressive Web App (PWA)
- Dark mode and customizable themes

## Tech Stack
Frontend:
- HTML
- CSS
- Vanilla JavaScript
- Chart.js

Backend:
- Node.js
- Express.js

Database:
- MongoDB Atlas

Deployment:
- Render
